üìã What We Check (Governance Rules)

Performance Rules

  • Auto Date/Time features must be disabled (improves load speed)
  • Efficient relationship structures
  • Optimized data models

Logic & Quality Rules

  • No bidirectional relationships (prevents data integrity issues)
  • Proper measure documentation
  • Consistent naming conventions

Documentation Rules

  • Every measure must have a description
  • Report purpose documented
  • Data sources identified

Brand Standards

  • AB InBev official theme applied
  • Consistent color palette
  • Professional layout templates
