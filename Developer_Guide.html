  2. Under "Current File" section, click Data Load
  3. UNCHECK "Auto date/time"
  4. Click OK
  5. Save your file (Ctrl + S)

  3. If you see arrows on both ends ‚Üí bidirectional!
  4. Right-click the relationship ‚Üí Properties
  5. Change "Cross filter direction" to Single
  6. Click OK
